Lottie Animation

Stuff I learnt in Week 14.

    For week 14 I learnt how to implement a lottie animation

    Firstly a Lottie is a JSON-based animation file format that enables designers to ship animations on any platform as easily as shipping static assets. They are small files that work on any device and can scale up or down without pixelation
