Stuff I learnt in Week 5.

    Learnt The Basics of Javascript such as

  • How to create variables like string which can be
    let msg;
    msg ="Hello";
  • Objects and Arrays
    Where objects are "things" for example:
    || Example 'person' object
    var person = {
    name: 'Zac',
    age: 33,
    likesCoding: true
    };
    Where are Arrays are like storage holders to hold multiple items in one variable ,Example:
    var vacationSpots = ['Tokyo', 'Bali', 'Mexico City', 'Vancouver'];
    We can also take certain items for example:
    vacationSpots[2]; // returns 'Mexico City'
  • Lastly I also learn loops which are similar to C# for example:
    for (let step = 0; step < 5; step++) {
    // Runs 5 times, with values of step 0 through 4.
    console.log("Walking east one step");
    }
    The code above increases the step count to a max of 5
